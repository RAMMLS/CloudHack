# SRE Agent for Kubernetes - Backend

Интеллектуальный помощник для диагностики и управления Kubernetes кластерами.

## Функциональность

- **Мониторинг Kubernetes**: Состояние нод, подов, сервисов, деплойментов
- **Сбор метрик**: Интеграция с Prometheus для сбора метрик
- **Анализ логов**: Интеграция с Loki для анализа логов
- **Трассировка**: Интеграция с Jaeger для анализа распределенных трассировок
- **Анализ инцидентов**: Автоматический анализ и рекомендации по устранению проблем
- **Проверка здоровья**: Комплексная проверка состояния всех компонентов

## Быстрый старт

### Требования

- Docker и Docker Compose
- Доступ к Kubernetes кластеру
- kubeconfig файл
